With this software on-hand, data collected by carchips or driverights is just a click away.


DriveRight’s Fleet Management Software allows your computer to work with your vehicle.
______________________


• Strong reporting and an intuitive interface deliver information in a timely and understandable manner.

• Reports are summarized to give the manager the information needed at a glance.

• More detailed reports, such as exception reports, allow customized tracking of vehicle use and specific driver behavior.

• Powerful ODBC (Open Database Connectivity) compliant design lets you choose the database platform, including Microsoft Access, SQL Server,MySQL, Oracle 8i and 9i, and others.

• Intelligent archiving allows the manager to keep relevant data active while keeping the past safely stored.

• Flexible in acquiring and sharing data. In addition to support for wireless, remote, or Palm downloading, data can be shuttled via FTP; equally comfortable on a single computer or a network server.

Thoughtful Reports, Flexible Connectivity
• Generates HTML-based reports via the built-in e-mail utility or for posting on any web server.

With CarChip Fleet and CarChip Fleet with Alarm, you'll also get additional reports on engine performance logging and diagnostic trouble codes.

With DriveRight 600E and the optional GPS Module, you can export the data to Google Maps, Google Earth or Microsoft's MapPoint or Streets and Trips to map each trip, including all starts and stops.

The software's OBDC-compliant design means you can use your preferred database platform, whether it be Microsoft Access, MySQL, Oracle 9i, or another.

Includes software for both DriveRight600E and CarChip Fleet. If you are using DriveRight600E you will need to use only the DriveRight600E software CD. If you are using CarChip Fleet and CarChip Fleet with Alarm, you will need to use both programs. Includes power adapter and cable for use when downloading data from DriveRight 600E (add our optional USB-to-Serial Cable to connect to USB) and USB cable for downloading CarChip. For DriveRight 600E, data can also be downloaded using our optional Palm Download Kit or Remote Download Kit.
