The Ethernet/Modem Gateway EMG-202S is a next-generation VoIP Gateway that provides a cost-saving solution for small business/home users on their telecommunication needs. The EMG-202S follows the open standard RFC-3261 SIP protocol to make sure that users can easily install the device on most of the existing VoIP (Voice over IP) services. By using the state-of-the-art DSP (Digital Signal Processing) technology, the EMG-202S delivers outstanding voice quality that is very comparable to the PSTN voice quality.

With built-in LCD display, users can easily configure the EMG-202S for first a time installation in a few minutes. And besides the advanced VoIP functions in easy installation, EMG-202S also provides flexible call out function between PSTN call and VoIP call through Ethernet or modem connection. EMG-202S is your best VoIP solution in the next generation communication.

FEATURES:

Built-in modem function (optional)
Choose between PSTN call and VoIP call
Follows RFC-3261 SIP standard
Dynamic IP support (DHCP and PPPoE)
Remote software upgrade capability (via FTP)
Advanced Digital Signal Processing (DSP) technology to ensure superior audio quality
Support G.723.1 (6.3k/5.3k), G.729A/B, G.711 (A-law/U-law) voice codecs
Support Sillence Suppression, VAD (Voice Activity Detection), CNG (Comfort Noise Generation)
System status display on the LCD panel
Call with or without proxy server (direct IP dialing)
Provide easy configuration methods
Support RFC-3261, TCP/UDP/IP, RTP/RTCP, HTTP, ICMP, ARP, DNS, DHCP, NTP/SNTP, FTP, PPPoE protocols
Interoperable with most existing SIP VoIP devices (IP-phone gateway, proxy server)
The LAN/WAN port works for paralleled Ethernet cable and crossed Ethernet cable.

SPECIFICATIONS:

Phone Port
1 x RJ11 connects to a telephone set. This telephone set can make PSTN call through the line port or VoIP call through Ethernet or modem connection
Line Port
1 x RJ11 connects to the PsTN line.
Built-in V.90 modem function (optional)
Work between PSTN call and modem function
LAN Port
1 x RJ45 10/100 Base-T Ethernet, line auto-sending/switching
WAN Port
1 x RJ45 10/100 Base-T Ethernet, line auto-sensing/switching
Power over Ethernet 802.3af function (optional)
